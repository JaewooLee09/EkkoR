# EkkoR 🕒⚡

**EkkoR** is a Chrome extension that brings the power of **Chronobreak** to your YouTube experience.

Ever clicked a timestamp to see a specific part of a video, only to find yourself lost at the top of the page, far from the comment you were reading? **EkkoR** allows you to "rewind" your scroll position instantly to where you were before the jump.

## ✨ Features

- **Chronobreak Navigation**: Remembers your exact scroll position in the comment section.
- **Rewind Button**: A simplified, premium icon button appears at the bottom center after you click a timestamp.
- **Drag-to-Dismiss (Flick)**: Don't want to go back? Just grab the button and "throw" it off-screen to dismiss it.
- **Master Shortcut [R]**: Press the **R** key (matching Ekko's ultimate) to instantly return to your comments without using the mouse.
- **Time-Twisting Smoothness**: Smooth scrolling takes you back to exactly where you left off.
- **Smart Cleanup**: The timeline self-corrects—the button disappears if you manually scroll back or move to a different video.

## 🕒 The Story Behind the Name

The name **EkkoR** is inspired by the League of Legends champion **Ekko**, the Boy Who Shattered Time. His ultimate ability, **Chronobreak (R)**, lets him rewind time to return to a previous state. This extension gives you that same mastery over your browser's "timeline," ensuring you never lose your place in a deep comment dive.

## 🛠 Installation

1. Clone this repository or download the ZIP.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** (top right corner).
4. Click **Load unpacked** and select this folder.
5. You're ready to shatter time!
